Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag B (não semântica).
Nesta frase, tenho um termo em destaque usando a tag STRONG (semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semântico).
Nesta frase, temos um termo em itálico usando a tag EM (semântico)

Texto marcado

Podemos criar também um texto marcado usando a tag MARK (não semântico).
Para mudar de cor, usamos CSS, usando MARK STYLE
Dessa forma, cada vez que quisermos trocar a cor de uma marcação, teremos que reescrever MARK STYLE etc.
Uma outra maneira está no outro exemplo.

Texto grande e texto pequeno

Estamos criando um texto grande, usando a tag BIG, que é obsoleta, e um texto pequeno, usando a tag SMALL, que permanece em uso, nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído , usando a tag DEL, para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto para inserido, com a tag INS (semântica), para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado com a tag U (não semântica).

Texto sobrescrito

para inserir coisas do tipo x2+3, com a tag SUP

Texto subscrito
